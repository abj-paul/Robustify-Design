EnvGuard is a solution to protect a system from environmental deviation. It provides an intuitive view through which users can upload, modify, understand, robustify and explore design alternatives. We also provide gemini integration to make it easier to understand the generated designs.

1. Project Management

1.1. Project List

You can see existing lists of projects here. You can also create new project from here. We will look into it soon. Select on any project to open it.

1.2. Project Hub

Its a single-spot to manage details on the project, including requirements specification, robustification and exploration. You can upload and edit UML and LTL specifications here. You can also compile UML to visualize it. However, current version only compiles UML and does not consider LTL. You can similarly upload system specifications, as shown above. For safety property, LTL notation is the norm. And robustification process can be configured using a json format.

1.3. Robustification

Finally, user can click on "Robustification" to robustify the behavioral model against environmental deviations. Robustification will take some time, typically around a minute. A success message will appear once it completes.

1.4. Report

You can see the generated report here. Note that, all versions are stored here. Click on any report to see details on it. It will open an html page. You can share the html page across team members to easily communicate design alternatives. You can click on any to expand and see the designs. The collapsable view is presented to reduce the cognitive load as users explore the generated design alternatives. Observe that each design alternatives has complexity scores assigned to it. It ranks them based on simple to complex order. This way, its easier to explore design alternatives. For rare use cases, user can simply download the designs using browser's default print functionality. Reusing existing features like this reduces the burden on this application and keep it single-purpose.

1.5. Gemini Integration

To make it easier to explore design alternatives, we provide gemini integration. You can view all the generated designs here and use any to talk with gemini. After selecting a design, previous chats will load. You can text to chat. It takes around 2-5s for gemini to reply.

1.6. Project Update and Deletion

You can go back to project list by clicking on Project Hub in top left. You can click on Create New Project in order to create an new project. You can enter project name and description to create a new project. Note that multiple projects can have the same name, it is of no issue. And to modify project name/description or to delete a project, you can click Edit Project from Project Hub. You can modify project details or delete them here. After deletion, you will be redirected back to project list layout.

2. User Authentication and Authorization

Give username, password to login. And complete the captcha. Finally, user can logout. As for registration, you can fill up username, organization and password. Password must be at least 6 characters. And username must be unique.
